# 🍕 Pizza Runner SQL Case Study

This repository contains my solutions to the **Pizza Runner SQL Case Study**, part of the **8 Week SQL Challenge**.

The case study focuses on solving real-world business problems using SQL, including customer orders, runner performance, pizza deliveries, and business metrics.

## 🛠️ Skills Used

* SQL Querying
* INNER JOIN & LEFT JOIN
* Aggregate Functions
* GROUP BY & HAVING
* CASE Expressions
* Common Table Expressions (CTEs)
* Window Functions
* Date Handling
* Data Cleaning
